============================================================
  柳工(LiuGong)设备智能匹配系统 v2.0
============================================================

【文件夹说明】

  products/    ← 把柳工产品Excel丢到这里（支持多个文件）
  inquiries/   ← 把客户询盘Excel丢到这里
  outputs/     ← 自动生成的匹配结果会出现在这里
  engine/      ← 引擎代码（不要动）

【使用方法】

  方法1: 双击 "一键运行.bat"
  方法2: 命令行运行  python engine/run.py

【工作流程】

  1. 把柳工产品资料Excel放进 products/ 文件夹
     - 支持任意格式(引擎会自动识别列名)
     - 可以放多个文件,每次运行都会全部读取
     - 更新产品时替换文件即可

  2. 把客户询盘Excel放进 inquiries/ 文件夹
     - 引擎会自动识别设备名称、规格、参数、数量列
     - 支持多个询盘文件同时处理

  3. 双击 "一键运行.bat"
     - 自动导入产品库
     - 自动匹配所有询盘
     - 结果输出到 outputs/ 文件夹

【产品Excel建议格式】

  建议包含以下列(中英文均可,引擎自动识别):
  - 型号/Model (必须)
  - 类别/Category (如:挖掘机、装载机等)
  - 吨位/Tonnage
  - 斗容/Bucket
  - 马力/HP
  - 描述/Description

【询盘Excel建议格式】

  建议包含以下列:
  - 设备名称 (必须)
  - 规格型号
  - 技术参数
  - 数量

【注意事项】

  - 所有Excel必须是 .xlsx 格式
  - 文件名不要包含特殊字符
  - 不要打开正在处理的文件
  - 产品库每次运行都会重新读取

  如有问题,联系技术支持。
